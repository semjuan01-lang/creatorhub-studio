# CreatorHub Studio

> Create. Stream. Grow.

## 🚀 Sobre o projeto

O CreatorHub Studio é uma plataforma profissional para criação e transmissão de lives, inspirada em OBS Studio, Streamlabs e Canva, com foco em criadores de conteúdo que transmitem pelo celular e computador.
